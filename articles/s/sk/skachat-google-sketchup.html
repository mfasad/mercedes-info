Google SketchUp — легендарный инструмент для 3D-моделирования, который за 20 лет прошёл путь от простого редактора для архитекторов до полноценной платформы с поддержкой VR и облачных технологий. Сегодня программа разделена на несколько версий (SketchUp Free, SketchUp Pro, SketchUp Make), каждая из которых имеет свои нюансы скачивания и использования. В этой статье разберём, где легально скачать актуальные версии, как избежать вирусов при установке, и какие альтернативы стоит рассмотреть, если SketchUp не подходит под ваши задачи.

Важно понимать: с 2017 года Google продал права на SketchUp компании Trimble, и теперь официальный сайт — app.sketchup.com, а не google.com/sketchup, как многие привыкли. Это привело к путанице: пользователи ищут "скачать Google SketchUp", но находят устаревшие или пиратские версии. Мы поможем разобраться в актуальных источниках и избежать типичных ошибок.

Системные требования: проверьте совместимость перед скачиванием

Одна из частых ошибок пользователей — скачивание версии SketchUp, не соответствующей их "железу". Например, SketchUp Pro 2026 требует OpenGL 3.1+, а на старых ноутбуках с интегрированной графикой (Intel HD Graphics 4000) программа либо не запустится, либо будет тормозить. Ниже минимальные и рекомендуемые требования для актуальных версий:

  • 🖥️ Операционная система: Windows 10/11 (64-bit) или macOS 12 Monterey и новее. SketchUp Make 2017 ещё поддерживает Windows 7, но без обновлений безопасности.
  • 🧠 Процессор: Минимум 2-ядерный 2 ГГц+, рекомендуется 4 ядра 3 ГГц+ (например, Intel i5/Ryzen 5).
  • 🎮 Видеокарта: 1 ГБ VRAM с поддержкой OpenGL 3.1. Для рендеринга в V-Ray или Enscape нужны 4 ГБ VRAM+ (например, NVIDIA GTX 1650).
  • 💾 ОЗУ: Минимум 8 ГБ, для сложных проектов — 16 ГБ+. При 4 ГБ SketchUp будет постоянно "подвисать".

Критическая особенность: На Mac с чипами Apple M1/M2 SketchUp Pro 2023/2026 работает только через Rosetta 2 (native-версия для ARM пока в разработке). Это означает, что производительность может упасть на 15–20% по сравнению с Intel-Mac или Windows-ПК.

    • ✅ SketchUp Pro (основное приложение)

    • 🚫 Нет технической поддержки (последнее обновление — 2017 год).
    • 🚫 Нет экспорта в .STL (важно для 3D-печати).
    • 🚫 Нет интеграции с Trimble Connect (облачное хранилище).
    • ✅ Поддерживает плагины (например, Fredo6’s Tools).
    • ✅ Работает офлайн без интернета.

    Скачать SketchUp Make 2017 можно здесь (выберите SketchUp Make 2017 (64-bit)). Установка аналогична Pro-версии, но без ввода лицензионного ключа.

    Как обойти ограничение на экспорт в STL?

    ⚠️ Внимание: На macOS SketchUp Make 2017 может не запускаться из-за отсутствия поддержки 32-битных приложений в новых версиях системы. В этом случае используйте SketchUp Free в браузере или виртуальную машину с Windows.

    Частые ошибки при скачивании и установке SketchUp

    Даже опытные пользователи сталкиваются с проблемами при работе с SketchUp. Вот самые распространённые ошибки и способы их решения:

    • 🔄 "SketchUp не запускается после установки":

      Причина: отсутствует Microsoft .NET Framework 4.8 или устаревшие драйверы видеокарты. Решение: обновите драйверы через NVIDIA GeForce Experience или AMD Adrenalin, затем установите .NET Framework с официального сайта.

    • 🖼️ "Модели отображаются без текстур":

      Причина: в настройках графики (Window → Preferences → OpenGL) отключено "Use Hardware Acceleration". Включите эту опцию и перезапустите программу.

    • 📥 "Не могу скачать SketchUp Free — пишет 'Your browser is not supported'":

      Решение: используйте Google Chrome или Microsoft Edge (последние версии). SketchUp Free не поддерживает Safari и устаревшие браузеры.

    • 🔑 "Закончился триал SketchUp Pro — как продлить":

      Официально продлить триал нельзя, но можно создать новый аккаунт Trimble с другим email. Альтернатива — перейти на SketchUp Free или купить лицензию со скидкой для студентов (~$55/год).

    Если после установки SketchUp тормозит при работе с большими моделями, попробуйте:

    1. Уменьшить количество полигонов (Window → Model Info → Statistics → Purge Unused).
    2. Отключить тени и стили (View → Face Style → Monochrome).
    3. Использовать плагин CleanUp3 для оптимизации геометрии.

    SketchUp и 3D-печать: как подготовить модель для печати

    SketchUp часто используют для создания моделей под 3D-печать, но здесь есть нюансы: программа по умолчанию создаёт неманифолдную геометрию (то есть модель с дырами или наложенными полигонами), что приводит к ошибкам при печати. Чтобы подготовить модель правильно:

    1. Проверка на "водонепроницаемость":

      Установите плагин Solid Inspector2 (бесплатно в Extension Warehouse). Он выделит красным все проблемные зоны.

    2. Экспорт в STL:

      В SketchUp Pro выберите File → Export → 3D Model → STL. В SketchUp Make 2017 сначала установите плагин SketchUp STL (см. спойлер выше).

    3. Настройки экспорта:

      В диалоге экспорта выберите:

      • ✅ Units: Millimeters
      • ✅ Binary format (меньше вес файла)
      • ✅ Export only current selection (если нужно выгрузить часть модели)

  • Постобработка в slicer:

    Откройте STL в Ultimaker Cura или PrusaSlicer и проверьте модель на ошибки ("Fix model").

    • Критическая рекомендация: При моделировании для 3D-печати всегда используйте группы или компоненты (Edit → Make Group/Component). Это предотвратит слипание геометрии и упростит редактирование.

    FAQ: ответы на частые вопросы о SketchUp

    Можно ли легально скачать SketchUp Pro бесплатно?

    Да, но только в виде 30-дневного триала с официального сайта. После истечения пробного периода программа перестанет работать. Бесплатные альтернативы: SketchUp Free (веб) или SketchUp Make 2017 (десктоп).

    Почему SketchUp тормозит на моём ПК?

    Чаще всего из-за:

    • Нехватки ОЗУ (нужно минимум 8 ГБ, лучше 16 ГБ+).
    • Слабой видеокарты (проверьте поддержку OpenGL 3.1+).
    • Слишком детализированной модели (используйте плагин CleanUp3 для оптимизации).

    Также попробуйте отключить View → Edge Style → Profiles — это ускоряет отображение.

    Как перенести лицензию SketchUp Pro на другой компьютер?

    Лицензия SketchUp Pro привязана к аккаунту Trimble, а не к ПК. Чтобы активировать её на новом устройстве:

    1. Деактивируйте лицензию на старом ПК (Help → License → Deauthorize).
    2. Установите SketchUp на новый ПК и войдите в тот же аккаунт Trimble.
    3. Активируйте лицензию через Help → License → Authorize.

    Обратите внимание: количество активаций ограничено (обычно 2–3 устройства).

    Где скачать плагины для SketchUp?

    Официальный источник — SketchUp Extension Warehouse. Там есть как бесплатные (Fredo6’s Tools, 1001bit Tools), так и платные плагины (V-Ray, Enscape).

    ⚠️ Внимание: Не скачивайте плагины с сторонних сайтов (например, sketchucation.com без проверки) — они могут содержать вредоносный код.

    Можно ли использовать SketchUp на планшете или смартфоне?

    Официальных мобильных версий SketchUp нет, но есть альтернативы:

    • SketchUp Viewer (iOS/Android) — только для просмотра моделей.
    • Shapr3D (iPad) — параметрическое моделирование с поддержкой Apple Pencil.
    • Onshape (браузер/мобильное приложение) — CAD-система с облачным хранилищем.

    Для полноценной работы на Android можно установить SketchUp Free через браузер Chrome в режиме "Десктопная версия".
